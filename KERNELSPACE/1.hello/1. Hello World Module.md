# Program 1 - Hello World Module

## Goal
Understand module_init(), module_exit(), insmod, rmmod, dmesg.

## Flow
userspace -> insmod -> init_module() -> kernel logs

## Interview Questions
Q: Difference between built-in and module?
A: Built-in is linked into vmlinux. Module is loaded dynamically.

Q: What is vermagic?
A: Kernel version compatibility string.

Q: Why __init?
A: Init memory can be freed after initialization.

Q: Why __exit?
A: Cleanup code executed during rmmod.

## Typical Code Walkthrough
module_init(my_init);
module_exit(my_exit);

## Commands
insmod hello.ko
rmmod hello
dmesg | tail
