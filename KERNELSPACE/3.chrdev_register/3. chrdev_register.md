# Program 3 - register_chrdev()

## Major / Minor

Major -> Driver
Minor -> Device instance

Example:
/dev/null  => major 1 minor 3
/dev/i2c-0 => major 89 minor 0

## VFS Flow
open()
 -> VFS
 -> file_operations.open()

read()
 -> VFS
 -> file_operations.read()
